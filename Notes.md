## Immutable and Mutability : 

- Mutable Data types 
    - List
    - Set
    - Dictionary 
    - Bytearray
    - Array

- Immutable Data types :
    - Integers
    - Floating-point numbers
    - Boolean
    - Strings
    - Tuples 
    - Frozen set

- Immutability : the references made in memory are never touched but new objects<br>
